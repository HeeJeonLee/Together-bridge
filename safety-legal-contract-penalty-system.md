# 1️⃣ 안전성 시스템 (Safety System)

## 🔐 사전 예방 (Pre-Matching Prevention)

### 1. 신원확인 & 검증

```
고객 가입 시:
├─ 필수 정보:
│  ├─ 실명 (주민등록증 인증)
│  ├─ 휴대폰 번호 (SMS 인증)
│  ├─ 주소 (실제 거주지 확인)
│  └─ 신용카드 (결제 수단 등록)
│
├─ AI 자동 검증:
│  ├─ 휴대폰 번호 진위 확인 (Twilio)
│  ├─ 주소 유효성 검사
│  └─ 중복 가입 확인 (동일 기기, IP)
│
└─ 금융 검증:
   ├─ Toss로 신용카드 소액 거래 (소액 인증)
   └─ 결과: "본인 확인된 계정" 뱃지 부여

대행자 가입 시:
├─ 필수 정보 (위와 동일)
├─ 추가 정보:
│  ├─ 이력서 / 경력
│  ├─ 경험 분야
│  ├─ 신원확인 서류 (신분증 사진)
│  └─ 은행 계좌 (실명 확인)
│
├─ 수동 검증 (AI 분석 후):
│  ├─ 의심 계정 플래그 (나이, 지역 이상)
│  ├─ 경력 검증 (과장 여부)
│  └─ 이전 사건 조회 (경찰청 조회 연계*)
│
└─ 여성 안전 특별 절차:
   ├─ 여성 대행자: 추가 신원 검증
   ├─ 성범죄 전력 조회 (경찰청 협력)
   ├─ 신용평점 확인
   └─ 참고인 제출 (필요시)

*주의: 경찰청 조회는 법적으로 "범죄인 조회" 범위 내에서만 가능
→ 당근마켓, 우버 등도 유사하게 운영

검증 수준:
├─ Level 1 (기본): 모든 사용자 자동
├─ Level 2 (강화): 의심 계정 수동
└─ Level 3 (철저): 여성안전 대행자 추가 검증
```

### 2. 여성 안전 시스템 (차별화)

```
시스템 1: 자동 배정
├─ 여성 고객이 "여성 대행자 선호" 체크
├─ AI가 자동으로 여성만 매칭
├─ 남성은 선택 불가
└─ 자동화율: 100%

시스템 2: 신뢰도 강화
├─ 여성 대행자에 추가 검증:
│  ├─ 신원확인 서류 심사
│  ├─ 경력 확인 (참고인 등)
│  ├─ 신용평점 확인
│  ├─ 범죄경력 확인
│  └─ 정기 재검증 (6개월마다)
│
└─ 여성 대행자 뱃지: "검증완료 여성"

시스템 3: 동반 서비스
├─ 여성이 동행이 필요한 경우:
│  ├─ 고객도 여성 / 고객의 여성 동반자
│  └─ 이중 검증으로 안전 강화
│
└─ 예: "부모님 병원 동행" 고객의 딸도 함께

시스템 4: 실시간 안전 기능
├─ SOS 버튼 (즉시 119 신고)
├─ 위치 공유 (신뢰할 사람에게)
├─ 실시간 통화 기록
├─ 종료 후 5분 내 확인 메시지
└─ 응답 없으면 자동으로 경찰 신고

결과: 여성에게 최고 수준의 안전 제공 ✅
```

### 3. CCTV & 증거 기록

```
권장 사항:
├─ 모든 거래에 CCTV 촬영 권장
│  └─ "안전한 장소에서 진행하세요"
│
├─ 거래 전 양쪽이 확인:
│  ├─ "촬영에 동의하시나요?" (앱 내 버튼)
│  └─ 동의 = 법적 증거로 인정
│
└─ 문제 발생 시:
   ├─ CCTV 영상 수집
   ├─ 경찰 신고 시 제출
   └─ 우리의 강력한 증거

위험한 장소 필터:
├─ 자동 필터링:
│  ├─ 숲, 외진 곳 (GPS 기반)
│  ├─ 밤 11시 이후 오지
│  └─ 경고 메시지 표시
│
└─ 위험 평가:
   ├─ 녹색: 안전함 ✅
   ├─ 노랑: 주의 ⚠️
   └─ 빨강: 진행 불가 ❌
```

# 2️⃣ 법적 계약 체계 (Legal Framework)

## 📜 이용약관 구조

### 1. 기본 이용약관

```
주요 조항:

1조: 목적
├─ 안전하고 투명한 매칭 서비스 제공
├─ 양 당사자의 신뢰 기반 거래
└─ 분쟁 해결 및 배상

2조: 서비스 범위
├─ 우리의 역할: "중개 및 매칭만 제공"
├─ 거래는 개인 간 계약
└─ 우리는 거래에 직접 참여하지 않음
    (중요: 법적 책임 제한)

3조: 사용자의 책임
├─ 정확한 정보 제공
├─ 안전한 환경에서 거래
├─ 촬영 동의 및 증거 남기기
└─ 약관 위반 시 계정 중지

4조: 금지 사항
├─ 성범죄, 폭력, 재산범죄
├─ 사기, 기만, 허위 정보
├─ 약물, 무기 거래
├─ 개인정보 유출
└─ 혐오 및 차별

5조: 플랫폼의 책임 범위
├─ ✅ 우리가 책임지는 것:
│  ├─ 신원확인 체계 운영
│  ├─ 신뢰도 평가 시스템
│  ├─ 보험 제공
│  ├─ 사고 신고 체계
│  └─ 배상 중재
│
├─ ❌ 우리가 책임지지 않는 것:
│  ├─ 거래 자체의 성공 여부
│  ├─ 개별 거래의 질
│  ├─ 당사자 간의 신뢰도
│  └─ 계약 이행 (매칭 외)

6조: 매칭 거절 패널티
├─ (자세한 내용은 별도 섹션)
└─ ...

7조: 분쟁 해결
├─ 1차: 플랫폼 중재 (무료)
├─ 2차: 소비자 분쟁 조정 (한국소비자원)
├─ 3차: 소송 (서울중앙지법)
└─ 중재비용: 패자 부담
```

### 2. 구독 이용계약서 (선택사항)

```
프리미엄 구독 상품 (향후):
├─ VIP 대행자 배치 (월 5,000원)
├─ 우선순위 매칭 (1회 2,000원)
└─ 자동갱신 구독 형태

계약 조항:

1. 구독 기간
├─ 월 1회 자동갱신
├─ 7일 전 취소 가능
└─ 취소 시 익월부터 정지

2. 환불 정책
├─ 사용하지 않은 부분: 일할 계산 환불
├─ 사용한 부분: 환불 불가
└─ 취소 후 7일 이내 환불

3. 서비스 제공 실패
├─ "최선을 다하지만, 보장하지 않음"
│  (매칭 실패 가능성)
├─ 1개월 내 매칭 실패 시: 전액 환불
└─ 그 이후: 환불 불가

4. 해약 조항
├─ 사유:
│  ├─ 서비스 중단 (우리 책임)
│  ├─ 약관 위반 (사용자 책임)
│  └─ 합의 해약
│
└─ 배상:
   ├─ 우리 책임: 전액 환불 + α (배상금)
   └─ 사용자 책임: 환불 없음 + 계정 정지
```

### 3. 이용료 결제 약관

```
기본 구조:

1. 수수료 공시
├─ 거래액의 10-15% (전자상거래법 준수)
├─ 계약 전 명확히 표시
└─ 수정 시 2주 전 공지

2. 결제 방식
├─ 선결제 (거래 전 결제)
│  ├─ 고객이 플랫폼에 입금
│  ├─ 거래 완료 후 대행자에게 송금
│  └─ 안전 장치 (플랫폼이 보증)
│
└─ 혜택:
   ├─ 고객: 사기 방지
   ├─ 대행자: 미수금 없음
   └─ 플랫폼: 수수료 미리 수령

3. 거래 취소 및 환불
├─ 대행자가 수락 전: 즉시 환불
├─ 대행자가 수락 후, 고객 취소:
│  ├─ 대행자 보상금 (거래액 20%)
│  ├─ 플랫폼 수수료 (거래액 10%)
│  └─ 고객 환금 = 거래액 × 70%
│
└─ 대행자 취소: 별도 규정 (아래)

4. 분쟁 중 환불 대기
├─ 분쟁 발생 시:
│  ├─ 결제금: 플랫폼 에스크로우 보관
│  ├─ 중재 진행
│  └─ 판정 후 지급
│
└─ 기간: 최대 30일
```

# 3️⃣ 매칭 거절 페널티 시스템 (Penalty System)

## ⚠️ 왜 페널티가 필요한가?

```
문제점:

"AI가 완벽하게 매칭했는데
 대행자가 갑자기 거절하면?"

├─ 고객: 시간 낭비, 신뢰도 하락
├─ 플랫폼: 거래 성사율 저하
├─ 생태계: 신뢰도 붕괴 위험
└─ 결과: 사업 실패 가능

해결책:
"합리적인 수준의 페널티 시스템"
```

## 🎯 페널티 시스템 구조

### 1단계: 거절 사유 분류

```
허용되는 거절:
─────────────────────────────
1. 긴급 사유:
   ├─ 갑작스러운 질병
   ├─ 가족 응급 상황
   ├─ 심각한 사고
   └─ 페널티: 없음 (증명 필요)

2. 합리적 사유:
   ├─ 거래지가 너무 먼 경우 (30km+)
   ├─ 충돌하는 기존 약속 발견
   ├─ 소요 시간 오계산
   └─ 페널티: 낮음 (50%)

거절하면 안 되는 경우:
─────────────────────────────
1. 명백한 거절:
   ├─ 이유 없이 거절
   ├─ "기분 나쁨"
   ├─ "다른 거래 선택"
   └─ 페널티: 높음 (100%)

2. 차별 거절:
   ├─ 나이, 성별, 지역 이유
   ├─ 외모 이유
   ├─ 여성/남성 이유
   └─ 페널티: 극도로 높음 (200%+)

시스템: AI가 자동으로 분류
```

### 2단계: 페널티 설계

```
기본 구조:

거절 1회 (30일 내):
├─ 경고: "거절 기록" (앱에 표시)
├─ 패널티: 없음
└─ 신뢰도: -5점

거절 2회 (30일 내):
├─ 경고: "거절 기록 증가"
├─ 패널티:
│  ├─ 보증금 5만원 선취
│  └─ 신뢰도: -15점
└─ 결과: 신뢰도 80 이하 (매칭 제한)

거절 3회 (30일 내):
├─ 경고: "계정 검토"
├─ 페널티:
│  ├─ 보증금 10만원 선취
│  ├─ 7일 계정 정지
│  └─ 신뢰도: -30점 (급락)
└─ 결과: 자동 일시정지

거절 4회+ (30일 내):
├─ 경고: "계정 폐기 검토"
├─ 페널티:
│  ├─ 보증금 전액 선취 (통상 10-20만원)
│  ├─ 계정 영구 정지
│  ├─ 신뢰도: 0점 (최악)
│  └─ 블랙리스트 등록
└─ 결과: 서비스 이용 불가

특수 경우: 차별 거절
├─ 1회만 해도:
│  ├─ 경고: "차별 거절 기록"
│  ├─ 페널티:
│  │  ├─ 보증금 50만원 선취
│  │  ├─ 7일 계정 정지
│  │  └─ 신뢰도: -50점
│  │
│  └─ 결과: 거의 이용 불가
│
└─ 재발 시:
   ├─ 계정 영구 정지
   ├─ 경찰 신고 (혐오 기준)
   └─ 손배 청구 가능
```

### 3단계: 보증금 제도

```
대행자 가입 시:
├─ 초기 보증금: 10만원
│  ├─ 첫 달: 0회 거절 시 환불
│  ├─ 월 1-2회 거절: 유지
│  └─ 월 3회+ 거절: 일부 선취
│
└─ 사용 방식:
   ├─ 거절 페널티로 자동 선취
   ├─ 신뢰도 회복 시 환불
   ├─ 계정 정지 시 보관
   └─ 계정 폐기 시 몰수

신뢰도 회복 방법:
├─ 30일간 거절 없음 → +20점
├─ 90일간 거절 없음 → +50점
├─ 거절 페널티 받은 달의 다음달부터 카운트
└─ 신뢰도 90 이상 → 보증금 환불
```

### 4단계: 신뢰도 점수 시스템

```
신뢰도 계산:

초기값: 100점

증가 요소:
├─ 거래 1건 완료: +3점
├─ 좋은 평점 (4.5★ 이상): +5점
├─ 30일 거절 없음: +10점
└─ 최대: 150점 (우수 대행자)

감소 요소:
├─ 거절 1회: -5~50점 (사유별)
├─ 낮은 평점 (3.0★ 이하): -10점
├─ 취소 (고객 취소): -3점
└─ 최소: 0점 (블랙리스트)

신뢰도별 혜택/제한:

150-100점 (우수):
├─ ✅ 모든 기능 사용 가능
├─ ✅ VIP 매칭 우선
├─ ✅ 고료금 수수료 10% (기본 12%)
└─ ✅ 월 보너스 5,000원 (기본 앱 머니)

100-80점 (정상):
├─ ✅ 모든 기능 사용 가능
├─ ⚠️ 일부 매칭 제한 (긴급 건만)
└─ ℹ️ 기본 수수료 12%

80-60점 (주의):
├─ ⚠️ 매칭 건수 제한 (일 2건)
├─ ⚠️ 수수료 인상 (15%)
└─ ⚠️ VIP 제외

60-40점 (경고):
```

# 6️⃣ 신뢰도 평가 시스템 (Trust Score)

## ⭐ 평점 & 신뢰도

### 1. 평점 시스템 (Stars)

```
대행자 평점 (고객이 부여):

1★: 매우 불만족
├─ 서비스 미이행
├─ 태도 불량
└─ 재거래 의사 없음

2★: 불만족
├─ 여러 문제 발생
├─ 예상과 다름
└─ 거래 완료되었으나 문제 있음

3★: 보통
├─ 서비스 완료했으나 미흡
├─ 태도는 괜찮음
└─ 필요하면 다시 할 것

4★: 좋음
├─ 서비스 잘 완료
├─ 태도 친절
└─ 다시 거래 의사 있음

5★: 매우 좋음
├─ 서비스 뛰어남
├─ 태도 최고
└─ 추천하고 싶음

평점 계산:
└─ 지난 3개월 거래 평균
   ├─ 1건 거래: 즉시 반영
   ├─ 2건 이상: 평균 계산
   └─ 항상 최신 상태 유지

시각화:
├─ 4.5★ 이상: 녹색 (우수)
├─ 3.5~4.4★: 노랑 (좋음)
├─ 2.5~3.4★: 주황 (보통)
└─ 2.4★ 이하: 빨강 (주의)
```

### 2. 신뢰도 점수 (Trust Score)

```
신뢰도 점수: 0-150점

요소별 가중치:

A. 평점 (40점):
├─ 5★ = 40점
├─ 4★ = 32점
├─ 3★ = 24점
├─ 2★ = 16점
└─ 1★ = 8점

B. 거래 건수 (20점):
├─ 50건 이상 = 20점
├─ 30-49건 = 15점
├─ 10-29건 = 10점
├─ 5-9건 = 5점
└─ 1-4건 = 0점

C. 거절률 (20점):
├─ 0% = 20점
├─ 1-5% = 15점
├─ 6-10% = 10점
├─ 11-20% = 5점
└─ 21%+ = 0점

D. 신원확인 (15점):
├─ 완전 검증 = 15점
├─ 기본 검증 = 10점
├─ 미검증 = 0점
└─ 여성 특별검증 = +5점

E. 계정 나이 (10점):
├─ 1년 이상 = 10점
├─ 6-11개월 = 7점
├─ 3-5개월 = 5점
└─ 0-2개월 = 0점

공식:
신뢰도 = 평점(40) + 거래(20) + 거절(20) 
         + 신원(15) + 나이(10) + 보너스(-5)
= 최대 150점

실제 예시:

우수 대행자 (신뢰도 130):
├─ 평점 4.8★ (38점)
├─ 거래 60건 (20점)
├─ 거절 2% (18점)
├─ 여성 특별검증 (20점)
├─ 2년 경력 (10점)
├─ 보너스 (거절 없음 30일) (+4점)
└─ = 130점 (우수)

일반 대행자 (신뢰도 90):
├─ 평점 4.0★ (32점)
├─ 거래 25건 (10점)
├─ 거절 5% (15점)
├─ 기본검증 (10점)
├─ 6개월 경력 (7점)
├─ 보너스 (거절 10회) (-4점)
└─ = 90점 (정상)

주의 대행자 (신뢰도 50):
├─ 평점 2.5★ (20점)
├─ 거래 5건 (5점)
├─ 거절 20% (5점)
├─ 미검증 (0점)
├─ 2개월 경력 (0점)
├─ 보너스 (거절 20회) (-5점)
└─ = 50점 (주의)
```
